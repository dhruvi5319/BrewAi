**`Dockerfile`** — From TechArch §7.5, EXACT spec. Constraints: node:20-bookworm-slim ONLY, no Alpine, no curl/wget, no X-Frame-Options DENY:
```dockerfile
FROM node:20-bookworm-slim

WORKDIR /app

COPY package*.json ./
RUN npm ci

COPY . .
RUN npm run build

RUN mkdir -p /app/data

EXPOSE 3000

ENV NODE_ENV=production
ENV HOST=0.0.0.0
ENV PORT=3000
ENV DB_PATH=./data/brewai.db

CMD ["node", "server.js"]
```
