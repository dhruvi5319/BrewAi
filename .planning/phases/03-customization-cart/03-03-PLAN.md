### CartItem.tsx

**Purpose:** Renders one line item in the cart: drink name, customization summary, line total, quantity stepper, and remove button.

**Props:**
```typescript
interface CartItemProps {
  item: CartItem;  // from src/types/index.ts
}
```

**Customization summary helper** (produce the "Large · Oat · Iced · Vanilla Syrup" string):
```typescript
function buildSummary(customizations: CartItemCustomizations): string {
  const parts: string[] = [];
  if (customizations.size) parts.push(customizations.size);
  if (customizations.milk) parts.push(customizations.milk);
  if (customizations.temperature) parts.push(customizations.temperature);
  if (customizations.shots) parts.push(customizations.shots);
  if (customizations.addons.length > 0) parts.push(customizations.addons.join(', '));
  return parts.join(' · ');
}
```

Special instructions are NOT shown in the summary string per FRD F03 §Customization Summary Format.

**Quantity stepper behavior:**
- Decrement: if `qty > 1` → `updateQuantity(cartItemId, qty - 1)`; if `qty === 1` → `removeItem(cartItemId)` (removes item entirely per F3-03)
- Increment: `updateQuantity(cartItemId, qty + 1)`, capped at 10 (button disabled at 10)

**Line total:** `(item.unitPrice × item.quantity).toFixed(2)` formatted as `$XX.XX`

**Implementation sketch:**
```typescript
import { useCartStore } from '../../stores/cartStore';
import { Trash2 } from 'lucide-react';
import type { CartItem as CartItemType } from '../../types/index';

interface CartItemProps { item: CartItemType; }

export function CartItem({ item }: CartItemProps) {
  const { updateQuantity, removeItem } = useCartStore();
  const summary = buildSummary(item.customizations);
  const lineTotal = (item.unitPrice * item.quantity).toFixed(2);

  return (
    <div className="flex gap-3 py-4 border-b border-border last:border-0">
      <div className="flex-1 min-w-0">
        <p className="font-display text-primary truncate">{item.name}</p>
        {summary && <p className="text-secondary text-sm mt-0.5 truncate">{summary}</p>}
        <p className="text-accent text-sm mt-1">${lineTotal}</p>
      </div>
      <div className="flex items-center gap-2 shrink-0">
        <button
          onClick={() => item.quantity === 1 ? removeItem(item.cartItemId) : updateQuantity(item.cartItemId, item.quantity - 1)}
          aria-label={`Decrease quantity for ${item.name}`}
          className="min-h-[44px] min-w-[44px] flex items-center justify-center rounded-lg border border-border hover:border-accent text-primary"
        >−</button>
        <span className="w-6 text-center text-primary text-sm">{item.quantity}</span>
        <button
          onClick={() => updateQuantity(item.cartItemId, item.quantity + 1)}
          disabled={item.quantity >= 10}
          aria-label={`Increase quantity for ${item.name}`}
          className="min-h-[44px] min-w-[44px] flex items-center justify-center rounded-lg border border-border hover:border-accent text-primary disabled:opacity-40 disabled:cursor-not-allowed"
        >+</button>
        <button
          onClick={() => removeItem(item.cartItemId)}
          aria-label={`Remove ${item.name} from cart`}
          className="min-h-[44px] min-w-[44px] flex items-center justify-center text-secondary hover:text-error ml-1"
        ><Trash2 size={16} /></button>
      </div>
    </div>
  );
}
```

### CartDrawer.tsx

**Purpose:** Slide-over panel listing all cart items. Desktop: `fixed right-0 top-0 h-full w-96 bg-surface`. Mobile (`< md`): `fixed inset-0 w-full`. Closes on Escape key and backdrop click (desktop only — no visible backdrop on mobile full-screen).

**Reads from useCartStore:** `items`, `subtotal`, `isOpen`, `clearCart`, `closeCart`.

**Clear cart confirmation:** Use inline React state `const [confirmClear, setConfirmClear] = useState(false)`. When `confirmClear` is true, show "Remove all items?" with "Cancel" and "Clear All" buttons. Do NOT use `window.confirm` (per constraints).

**Empty state:** When `items.length === 0`, show a message "Your cart is empty" and a "Browse Menu" link/button that calls `closeCart()` (the user is already on the menu page in Phase 3; Phase 4 adds routing).

**Place Order button:** Always visible in footer; `disabled` and `aria-disabled="true"` when `items.length === 0`. In Phase 3, clicking Place Order does nothing (Phase 4 wires the API call) — add a `// TODO Phase 4: submit order` comment.

**Responsive layout:**
```typescript
// Outer overlay (backdrop) — only on desktop md+
<div
  className="hidden md:block fixed inset-0 bg-canvas/50 z-40"
  onClick={closeCart}
  aria-hidden="true"
/>

// Drawer panel
<div
  role="dialog"
  aria-modal="true"
  aria-label="Your cart"
  className={`fixed top-0 right-0 h-full w-full md:w-96 bg-surface z-50 flex flex-col shadow-2xl transition-transform duration-200 ${isOpen ? 'translate-x-0' : 'translate-x-full'}`}
>
  {/* Header */}
  <div className="flex items-center justify-between p-4 border-b border-border">
    <h2 className="font-display text-xl text-primary">Your Cart</h2>
    <button onClick={closeCart} aria-label="Close cart" className="p-2 text-secondary hover:text-primary min-h-[44px] min-w-[44px]">✕</button>
  </div>

  {/* Content */}
  <div className="flex-1 overflow-y-auto p-4">
    {items.length === 0 ? (
      <div className="flex flex-col items-center justify-center h-full gap-4">
        <p className="text-secondary">Your cart is empty</p>

            aria-label="Clear all items from cart"

            <p className="text-sm text-primary mb-3">Remove all items from your cart?</p>
            <div className="flex gap-2">
              <Button variant="danger" size="sm" onClick={() => { clearCart(); setConfirmClear(false); }}>Clear All</Button>
              <Button variant="ghost" size="sm" onClick={() => setConfirmClear(false)}>Cancel</Button>
            </div>
          </div>
        )}
      </>
    )}
  </div>

  <div className="p-4 border-t border-border">
    <div className="flex justify-between mb-4">
      <span className="text-secondary">Subtotal</span>

    </div>
    <Button
      variant="primary"
      disabled={items.length === 0}
      aria-disabled={items.length === 0}

      className="w-full"
    >
      Place Order
    </Button>
  </div>
```

**Escape key handler** (useEffect):
```typescript
useEffect(() => {
  if (!isOpen) return;
  const handleKeyDown = (e: KeyboardEvent) => { if (e.key === 'Escape') closeCart(); };
  document.addEventListener('keydown', handleKeyDown);
  return () => document.removeEventListener('keydown', handleKeyDown);
}, [isOpen, closeCart]);
```

```typescript
}, [isOpen]);
```
