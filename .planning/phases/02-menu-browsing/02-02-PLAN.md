<task type="auto">
  <name>Task 1: ProductCard component</name>
  <files>src/components/menu/ProductCard.tsx</files>
  <action>
Create src/components/menu/ProductCard.tsx.

From FRD § F01 Product Card Layout:
```
┌─────────────────────────────┐
│  [Category Badge]           │
│                             │
│  Drink Name                 │  <- Playfair Display, 18px
│  Short description text     │  <- Inter, 14px, 2-line clamp
│  clamped to two lines here  │
│                             │
│  $4.50              [CTA]   │  <- price amber, CTA = primary Button
└─────────────────────────────┘
Card dimensions: full-width in grid cell; min-height 160px
```

Props interface:
```typescript
interface ProductCardProps {
  item: MenuItem;
  onCustomize: (item: MenuItem) => void;
  onAddToCart: (item: MenuItem) => void;
}
```

Implementation:
1. Outer wrapper: `<div className="flex flex-col justify-between bg-surface border border-border rounded-card p-4 min-h-[160px] hover:bg-surface-raised hover:border-border-hover transition-colors duration-150 ease-in-out">`
2. Top section `<div className="flex flex-col gap-2">`:
   - Category Badge: `<Badge variant="accent">{item.category}</Badge>` (import Badge from ../ui)
   - Drink name: `<h3 className="font-display text-lg font-semibold text-primary leading-snug">{item.name}</h3>`
     - font-display = Playfair Display as configured in tailwind.config.ts
   - Description: `<p className="text-sm text-secondary line-clamp-2 leading-relaxed">{item.description}</p>`
3. Bottom section `<div className="flex items-center justify-between mt-4">`:
   - Price: `<span className="text-accent font-semibold text-base">${item.basePrice.toFixed(2)}</span>`
   - CTA Button: if `item.hasCustomizations` is true, render `<Button variant="primary" size="sm" onClick={() => onCustomize(item)}>Customize</Button>`; otherwise render `<Button variant="primary" size="sm" onClick={() => onAddToCart(item)}>Add to Cart</Button>`
4. Import: `import { MenuItem } from '../../types/index'` and `import { Button, Badge } from '../ui'`
5. Export: `export function ProductCard({ item, onCustomize, onAddToCart }: ProductCardProps)`
6. NO raw hex values. NO inline styles. Only Tailwind utility classes.
7. Do NOT add Framer Motion wrappers here — MenuPage will wrap cards in motion.div (Phase 5 will wire stagger animation; for now just render the card content without motion).

Note: `hasCustomizations` is a boolean on MenuItem (from TechArch § 4.2). The `Badge` component from Phase 1 should accept `variant="accent"` per FRD § F00 primitive spec.
